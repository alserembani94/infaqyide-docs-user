# Teknologi

Berikut adalah _tech stack_ yang diguna pakai dalam pembangunan infaqYIDE Admin.

1. [NextJS](https://nextjs.org/docs/getting-started) - Sebuah _framework_ React, dilengkapi dengan teknologi _server-side rendering_ (SSR) dan _static side generation_ (SSG).
2. Github - Pengurusan _project versioning_.
3. AWS Amplify - _Deployment_ projek, menggunakan _CI/CD Automation_ melalui Github.
